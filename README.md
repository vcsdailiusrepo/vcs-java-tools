[Maven cendral repository](https://central.sonatype.com/artifact/io.github.vcsdailiusrepo/vcs-java-tools)

```xml
        <groupId>io.github.vcsdailiusrepo</groupId>   
        <artifactId>vcs-java-tools</artifactId>
        <version>1</version>
    </dependency>
    
```
